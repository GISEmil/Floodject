    What is "Flooding"?

    This is a page built as part of our master thesis for Geoinformatics at Aalborg University, campus Copenhagen. The project has been created to do a variety of cool things, but mostly to enable some users to do what they need to do without needing a certain set of skills or knowledge.

    Designed for developers

    This program has been designed to simplify the modelling of a flood as much as possible. It still requires techincal knowledge though.½

    Time saver

    The project has been developed to try and make the flood modelling as quick and efficient as possible.

    Advanced

    Created to enable a relatively advanced modelling of floods.

    Features

    • Robust
    • Simple
    • Written mostly in Python
    • Built with Open-source technologies:
    • GRASS
    • GDAL/OGR
    • Flask
    • Compatible with all modern browsers
